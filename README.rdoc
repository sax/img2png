== Img2Png

Img2Png is a small Sinatra application that uses RMagick to convert images into
the PNG format.

== Usage

http://png.heroku.com/ - about page
http://png.heroku.com/http/path/to/gif/i.png
http://png.heroku.com/https/path/to/gif/i.png
